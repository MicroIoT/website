# 当前设备信息

查询当前登录设备的详细信息。

``` HTTP
GET https://www.microiot.top/server/devices/me
```
## 访问控制

| 访问用户角色 | 是否需要认证                                 |
| :----------- | :------------------------------------------- |
| 设备         | 是，使用Bearer Token认证，具体信息见登录授权 |

## 请求体

无

## 响应

| 状态 | 类型          | 描述           |
| ---- | ------------- | -------------- |
| 200  | [Device](../device/adddevice.md#device) | 设备信息 |
