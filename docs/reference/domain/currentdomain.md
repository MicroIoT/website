# 当前领域信息

登录用户当前工作的领域信息。

``` HTTP
GET https://www.microiot.top/server/domains
```

## 访问控制

| 访问用户角色           | 是否需要认证                                 |
| :--------------------- | :------------------------------------------- |
| 系统管理员或区域管理员 | 是，使用Bearer Token认证，具体信息见[REST API认证](../api.md) |

## 请求体

无

## 响应

| 状态 | 类型          | 描述           |
| ---- | ------------- | -------------- |
| 200  | [Domain](adddomain.md#domain) | 领域信息 |
